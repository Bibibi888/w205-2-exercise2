Hello and welcome to TweetWordCount!

In order to run this program, please take the provided variables_dummy.rc file, modify it to include your user credentials (password is not necessary), and source the file. If you haven't sourced a file before, simply run 'source <filename>' and the file will load all credentials into your environment.

Once you have all credentials loaded into your environment, please change your directory to extweetwordcount and run "sparse run". No special launch parameters are required.

Enjoy your tweet stream!
